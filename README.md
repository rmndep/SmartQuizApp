# Smart Quiz App

A simple quiz application for Android that allows users to test their knowledge on various topics.

## Features

*   **User Authentication:** Users can enter their name before starting the quiz.
*   **Topic Selection:** Choose from a variety of quiz topics, including Android, Java, and Kotlin.
*   **Timed Questions:** Each question is timed to add a challenging element to the quiz.
*   **Scoring:** The app calculates and displays the user's score at the end of the quiz.
*   **High Score Tracking:** The app saves and displays the high score for each topic.
*   **Answer Review:** Users can review their answers after completing the quiz to see which questions they got right and wrong.
*   **Randomized Questions:** Questions are shuffled each time a new quiz is started.

## Technologies Used

*   **Language:** Java
*   **IDE:** Android Studio
*   **UI:** XML

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Android Studio
*   An Android device or emulator

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/rmndep/SmartQuizApp.git
    ```
2.  Open the project in Android Studio.
3.  Build and run the app.
